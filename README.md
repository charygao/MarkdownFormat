# MarkdownFormat README

MD 格式化

我自己用的挺好的，主要用于格式化 MD 文档如果你有其他的想法可以告诉我。

* 标题前后加入空行
* 文档最末尾一空行
* 中英文符号替换
* 汉字后的标点符号，转成全角符号。
* 全角数字 转半角
* 全角英文和标点 转半角
* 汉字后的标点符号，转成全角符号。
* 修复 markdown 链接所使用的标点
* 汉字与其前后的英文字符、英文标点、数字间增加空白。
* 保持代码块不受影响，代码块前后增加一空行 ` ``` `
* 保持链接原样不受影响  `()[]`
